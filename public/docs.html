Introduction

The PageSnap API allows you to programmatically capture high-quality screenshots of websites. Whether you need desktop previews or mobile-first audits, our API handles the browser infrastructure so you don't have to.

Base URL: https://57b0309679f05bbe7ed77f0c3ddd11e0.ctonew.app/v1

Authentication

All API requests must include your API key in the X-API-Key header. You can get your API key by subscribing to a plan. 

curl -H "X-API-Key: your_api_key_here" \
     https://57b0309679f05bbe7ed77f0c3ddd11e0.ctonew.app/v1/health

Take a Screenshot

Render a website and capture its visual state.

POST /screenshot

Request Body

Parameter Type Description
url string Required. The website URL to capture.
preset string Option: desktop, tablet, mobile. Default: desktop.
fullPage boolean Capture the entire page scroll. Default: false.
format string png, jpeg, webp. Default: png.

Example Request

curl -X POST \
     -H "Content-Type: application/json" \
     -H "X-API-Key: ps_live_12345" \
     -d '{
       "url": "https://google.com",
       "preset": "mobile",
       "fullPage": true
     }' \
     https://57b0309679f05bbe7ed77f0c3ddd11e0.ctonew.app/v1/screenshot
